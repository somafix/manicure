# 💅 Manicure Booking Bot

Бот для автоматизації запису на манікюрні послуги через Telegram. Дозволяє клієнтам самостійно обирати послугу, дату та вільний час, а адміністратору — отримувати сповіщення про нові записи.

## 🌟 Основні можливості

* **📅 Гнучкий запис:** Клієнти можуть обрати послугу та вільне вікно на найближчі 4 дні.
* **⏰ Контроль робочого часу:** Бот враховує робочі години (08:00–18:00) та обідню перерву (13:00–14:00).
* **🛠 Управління послугами:** Перегляд списку доступних послуг з їх тривалістю та ціною.
* **📋 Мої записи:** Перегляд власних бронювань та можливість скасування (не пізніше ніж за 2 години до початку).
* **👑 Адмін-панель:** Перший користувач, що натисне `/start`, автоматично стає адміністратором і отримує сповіщення про кожну нову дію.
* **📊 Моніторинг:** Швидкий перегляд вільних та зайнятих слотів на сьогодні.

---

## 🛠 Технологічний стек

* **Мова:** Python 3.x
* **API:** Telegram Bot API (через `requests`)
* **База даних:** SQLite (модуль `db.py`)
* **Час:** `datetime` для валідації робочих годин та слотів.
